# 主要画面ワイヤーフレーム

## 1. 目的

本ドキュメントは、音声AI見積作成システムのMVPで重要となる主要画面のワイヤーフレームを定義する。

画面遷移は、見積編集画面を起点とする。ユーザーは見積編集画面で顧客を選択し、その見積に対して明細をボイス入力する。

## 2. 対象画面

| 画面ID | 画面名 | 優先度 | 主な端末 |
| --- | --- | --- | --- |
| S-005 | 見積編集画面 | 最重要 | スマホ、PC |
| S-003 | 明細ボイス入力画面 | 高 | スマホ |
| S-004 | AI解析結果確認画面 | 高 | スマホ、PC |
| S-016 | 出力設定画面 | 高 | スマホ、PC |
| S-019 | 打ち合わせ録音画面 | 別オプション | スマホ、PC |

## 3. S-005 見積編集画面

### 3.1 目的

見積作成の中心画面。顧客選択、見積基本情報入力、明細確認、明細ボイス入力開始、業者指示事項編集、保存、出力を行う。

### 3.4 主な操作

- 顧客選択、顧客変更
- 見積基本情報入力
- 明細手入力追加
- 明細ボイス入力開始
- 打ち合わせ録音開始（別オプション有効時のみ）
- AI解析結果から採用された明細の編集
- 明細の場所、品目、数量、単位、単価、金額、備考、業者指示事項の編集
- 明細のドラッグアンドドロップ並び替え
- スマートフォンでの上下移動ボタンによる並び替え
- 見積全体の業者指示事項編集
- 明細ごとの業者指示事項編集
- 下書き保存
- 出力設定画面へ遷移

### 3.5 重要ルール

- 顧客未選択でも下書き作成は可能とするかは要検討。ただしMVPでは顧客選択後に明細ボイス入力する流れを基本とする。
- 顧客向け備考には「PDF表示」ラベルを付ける。
- 業者指示事項には「内部用・PDF非表示」ラベルを付ける。
- 業者指示事項はPDFプレビューにもPDF出力にも表示しない。
- 明細ボイス入力は同一見積に対して複数回実行できる。

## 4. S-003 明細ボイス入力画面

### 4.1 目的

選択中の見積に対して、音声で明細情報を入力する。音声は文字起こしされ、AI解析結果確認画面へ送られる。

### 4.2 スマートフォンワイヤーフレーム

```text
┌────────────────────────────┐
│ 明細ボイス入力        閉じる │
├────────────────────────────┤
│ 見積: 外壁塗装工事          │
│ 顧客: 田中太郎 様            │
│                              │
│ 話してください                │
│ ┌────────────────────────┐  │
│ │        録音待機          │  │
│ │      00:00 / 03:00      │  │
│ └────────────────────────┘  │
│                              │
│ [● 録音開始]                 │
│ [停止] [再録音]              │
│                              │
│ 入力例                       │
│ 外壁120平米、足場あり、      │
│ 高圧洗浄、下塗り、中塗り、   │
│ 上塗り、北側の養生注意       │
│                              │
│ 文字起こし結果               │
│ ┌────────────────────────┐  │
│ │ 外壁120平米、足場あり... │  │
│ └────────────────────────┘  │
│                              │
├────────────────────────────┤
│ [AI解析へ進む]               │
└────────────────────────────┘
```

### 4.3 主な操作

- 録音開始
- 録音停止
- 再録音
- 文字起こし結果の編集
- AI解析へ進む
- 見積編集画面へ戻る

### 4.4 入力対象

音声で入力する主な内容:

- 場所
- 品目
- 数量
- 材料グレード
- 明細備考
- 業者指示事項
- 現場注意事項
- 未確定事項

### 4.5 重要ルール

- 音声入力は明細入力の補助であり、解析結果は自動確定しない。
- 文字起こし結果はユーザーが修正できる。
- 顧客情報、件名、見積日、有効期限、担当者、顧客向け備考などのヘッダー情報はAI入力対象にしない。
- 明細の単位はAIが判断せず、選択した単価マスター品目の単位を表示する。
- 業者指示に該当しそうな文言は、AI解析結果確認画面で内部情報として分離する。

## 5. S-004 AI解析結果確認画面

### 5.1 目的

文字起こし結果からAIが抽出した明細候補、確認事項、業者指示事項候補を確認し、採用する項目だけを見積編集画面へ反映する。

### 5.2 PCワイヤーフレーム

```text
┌─────────────────────────────────────────────────────────────────────┐
│ AI解析結果確認                                                       │
├───────────────────────┬─────────────────────────┬───────────────────┤
│ 文字起こし結果         │ 明細候補                 │ 確認事項           │
│                       │                         │                   │
│ 外壁120平米、足場あり │ □ 外壁 / 足場設置 120m2  │ □ 塗料グレード確認 │
│ 高圧洗浄、下塗り...   │ □ 外壁 / 高圧洗浄 120m2  │ □ 下地補修の有無   │
│ 北側の養生注意...     │ □ 外壁 / 下塗り 120m2    │ □ 駐車場所確認     │
│                       │ □ 外壁 / 中塗り 120m2    │                   │
│                       │ □ 外壁 / 上塗り 120m2    │ 業者指示事項候補   │
│                       │                         │ 内部用・PDF非表示  │
│                       │ 単価マスター品目候補     │ □ 北側養生注意     │
│                       │ 一致度 92%               │ □ 搬入経路確認     │
│                       │                         │                   │
├───────────────────────┴─────────────────────────┴───────────────────┤
│ [戻る] [採用して見積編集へ反映]                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 5.3 スマートフォンワイヤーフレーム

```text
┌────────────────────────────┐
│ AI解析結果確認              │
├────────────────────────────┤
│ タブ: [明細候補] [確認] [指示]│
│                              │
│ 明細候補                     │
│ □ 足場設置                   │
│   場所: 外壁                  │
│   120 m2 × 900円             │
│   単価マスター一致度 92%     │
│                              │
│ □ 高圧洗浄                   │
│   120 m2 × 250円             │
│                              │
│ 確認事項                     │
│ ・塗料グレードが未指定です   │
│ ・下地補修の有無を確認       │
│                              │
│ 業者指示事項候補             │
│ 内部用・PDF非表示            │
│ □ 北側の養生注意             │
│ □ 搬入経路を事前確認         │
│                              │
├────────────────────────────┤
│ [戻る] [採用して反映]        │
└────────────────────────────┘
```

### 5.4 主な操作

- 明細候補を採用、除外
- 単価マスター品目候補を選択、変更
- 場所、品目、数量、備考を修正
- 単価マスター品目を選択し、マスター由来の単位、単価、金額を確認
- 確認事項を残す
- 業者指示事項候補を採用、除外
- 見積編集画面へ反映
- 明細ボイス入力へ戻る

### 5.5 重要ルール

- AI解析結果はユーザーが採用するまで見積に反映しない。
- AI解析結果確認画面では、顧客情報や見積ヘッダー情報の候補を表示しない。
- 業者指示事項候補は、顧客向け備考とは別枠で表示する。
- 業者指示事項候補には「内部用・PDF非表示」を表示する。
- 単位と単価は、選択した単価マスター品目に紐づく値を使用し、AIが判断しない。

### 6.5 重要ルール

- 業者指示用Excelでは、顧客向け金額を出すかどうかは別設定とする。

## 7. S-019 打ち合わせ録音画面（別オプション）

### 7.1 目的

営業担当者が顧客との打ち合わせ内容を録音し、文字起こし、要点、顧客要望、確認事項、見積反映候補として保存する。MVP本体ではなく、打ち合わせ録音オプションが有効な会社のみ表示する。

### 7.2 スマートフォンワイヤーフレーム

```text
┌────────────────────────────┐
│ 打ち合わせ録音        閉じる │
├────────────────────────────┤
│ 顧客: 田中太郎 様            │
│ 見積: 外壁塗装工事           │
│ 営業: 山田                   │
│                              │
│ □ 顧客へ録音同意を確認済み   │
│                              │
│ ┌────────────────────────┐  │
│ │        録音待機          │  │
│ │      00:00 / 30:00      │  │
│ └────────────────────────┘  │
│                              │
│ [● 録音開始]                 │
│ [停止] [一時保存] [再録音]   │
│                              │
│ 文字起こし結果               │
│ ┌────────────────────────┐  │
│ │ お客様は外壁色を...     │  │
│ └────────────────────────┘  │
│                              │
│ AI整理                       │
│ ・要点                       │
│ ・顧客要望                   │
│ ・確認事項                   │
│ ・見積反映候補               │
│                              │
├────────────────────────────┤
│ [営業メモ保存] [候補を反映]  │
└────────────────────────────┘
```

### 7.3 主な操作

- 録音同意確認
- 録音開始
- 録音停止
- 一時保存
- 再録音
- 文字起こし結果の編集
- AI要点整理
- 見積反映候補の採用

### 7.4 重要ルール

- 録音同意確認が未チェックの場合は録音開始できない。
- 打ち合わせ録音は営業メモとして保存し、顧客提出用PDFには出力しない。
- AIが抽出した見積反映候補は、ユーザーが採用するまで見積に反映しない。
- 打ち合わせ録音は顧客履歴と見積履歴から参照できる。
- オプション無効の会社では、見積編集画面と顧客詳細画面に打ち合わせ録音ボタンを表示しない。

## 8. 共通ラベル・表示ルール

### 8.1 情報種別ラベル

| ラベル | 意味 | 表示場所 |
| --- | --- | --- |
| PDF表示 | 顧客向けPDFに表示される情報 | 顧客向け備考 |
| PDF非表示 | 顧客向けPDFに表示されない情報 | 業者指示事項 |
| 内部用 | 社内・協力業者向け情報 | 業者指示事項、社内メモ |
| 要確認 | ユーザー確認が必要なAI抽出結果 | AI解析結果確認画面 |

### 8.2 ボタン配置

| 画面 | 主ボタン | 補助ボタン |
| --- | --- | --- |
| 見積編集画面 | 明細をボイス入力、出力設定へ | 打ち合わせ録音（別オプション）、手入力で明細追加、下書き保存 |
| 明細ボイス入力画面 | AI解析へ進む | 再録音、閉じる |
| 打ち合わせ録音画面 | 別オプション。営業メモ保存、候補を反映 | 一時保存、再録音、閉じる |
| AI解析結果確認画面 | 採用して反映 | 戻る |
| 出力設定画面 | 出力する、PDFプレビュー | 戻る |
