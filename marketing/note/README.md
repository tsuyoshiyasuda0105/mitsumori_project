## 投稿手順

1. `02_reform_construction_estimate_ai_note.md` を開く。
2. `noteタイトル` の下の1行を、noteのタイトル欄に貼り付ける。
3. `貼り付け本文` から本文末尾のリンクまでを、note本文欄に貼り付ける。
4. `ハッシュタグ` のタグを、noteのタグ欄または本文末尾に貼り付ける。
5. 投稿前チェックを確認してから公開する。

## 貼り付ける本文範囲

`# 貼り付け本文` の次の行から、本文末尾の「元の記事はこちらです。」のURLまで。

`# noteタイトル`、`# 貼り付け本文`、`# ハッシュタグ` の見出し自体は投稿本文に含めない。

## 投稿前チェック

- タイトル欄にタイトルが入っている
- 本文内のデモURLと元記事URLが開ける
- 表が崩れていない
- ハッシュタグが7個入っている
- frontmatterや作業用見出しを本文に貼っていない

## 動画の付け方

noteのテキスト記事では、YouTubeなどの動画URLを「埋め込み」で貼るのが安定です。

1. `C:\mitsumori_project\web\public\videos\voice-estimate-short.mp4` をYouTubeへ限定公開でアップロードする。
2. YouTubeの共有URLをコピーする。
3. note本文の「サンプル動画」セクションにある `【ここにYouTubeの限定公開URLを貼り付け】` と差し替える。
4. noteエディタの `+` から `埋め込み` を選び、同じURLを貼る。
