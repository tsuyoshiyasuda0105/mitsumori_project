# ショート動画制作・運用実行表

## 1. 目的

ChatGPT/Soraで、ボイス見積のショート動画を継続的に作成し、X、YouTube Shorts、Instagram Reelsへ展開する。

動画の主訴は以下の3点に統一する。

- 入力が苦手でも、音声で見積明細の下書きを作成できる
- 単価マスターで、担当者ごとの価格ばらつきを抑えられる
- 外部IDつきの見積データを、他システムへ出力・連携しやすくできる

旧訴求として、原価、業者指示事項、内部情報を顧客に出さない安心を主軸にしない。

## 2. 並列レーン

| レーン | 担当 | 成果物 | 完了条件 |
| --- | --- | --- | --- |
| Sora生成 | 動画担当 | 15秒・30秒の縦型動画 | 9:16、文字が読める、主訴3点のうち1点以上が明確 |
| 台本改善 | Codex | プロンプト、ナレーション、テロップ | そのままSoraに貼れる状態 |
| SNS投稿 | SNS担当 | X投稿文、固定ポスト、スレッド | LPまたはデモ導線が入っている |
| LP反映 | マーケ担当 | LP内の動画埋め込み位置案 | ファーストビュー直下、または機能説明直下に設置案 |
| 検証 | QA担当 | チェックリスト | 誤訴求、読みにくい字幕、不自然な画面を確認 |

## 3. 最初に作る動画

### 1本目: 入力が苦手でも音声で作成

目的: 初見の現場担当に刺す。

構成:

| 秒数 | 内容 | テロップ |
| --- | --- | --- |
| 0-3秒 | 現場担当がスマホに話す | 入力が苦手でも |
| 3-7秒 | 音声が文字起こしされる | 現場で話すだけ |
| 7-11秒 | 見積明細候補ができる | AIが明細を下書き |
| 11-15秒 | デモ画面とCTA | まずはデモで確認 |

投稿文:

```text
見積入力が苦手でも、現場で話せば始められる。

ボイス見積は、スマホに話した内容から見積明細の下書きを作る、建設業向けの音声AI見積作成システムです。

AIが作るのは下書き。
最後は担当者が確認します。

[デモURL]
```

### 2本目: 単価マスターで価格統一

目的: 経営者、事務担当、見積管理者に刺す。

構成:

| 秒数 | 内容 | テロップ |
| --- | --- | --- |
| 0-3秒 | 担当者ごとに違う見積単価 | 価格、バラついていませんか |
| 3-8秒 | Excel単価表を取り込む | Excel単価表をマスター化 |
| 8-13秒 | 明細に単価が反映される | 単価マスターで価格を統一 |
| 13-15秒 | CTA | デモで確認 |

投稿文:

```text
見積単価が担当者ごとに変わると、確認も修正も大変です。

ボイス見積では、既存のExcel単価表を単価マスターとして活用し、品目ごとの単位・単価をそろえる運用を目指せます。

[デモURL]
```

### 3本目: 他システムへ見積データ出力

目的: システム連携、業務効率化に関心がある層に刺す。

構成:

| 秒数 | 内容 | テロップ |
| --- | --- | --- |
| 0-3秒 | 見積を別システムへ転記している | 二重入力、まだ続けますか |
| 3-8秒 | 明細に外部IDが付く | 外部IDつき見積明細 |
| 8-13秒 | Excel/CSV/連携用データへ出力 | 他システムへ見積出力 |
| 13-15秒 | CTA | 連携前提で相談 |

投稿文:

```text
見積を作ったあと、施工管理・販売管理・会計システムへ同じ内容を入れ直していませんか。

ボイス見積は、外部IDつきの見積データを出力し、他システム連携につなげやすい設計を目指しています。

[デモURL]
```

## 4. Sora生成時の基本設定

| 項目 | 推奨 |
| --- | --- |
| 比率 | 9:16 |
| 尺 | 15秒または30秒 |
| トーン | 現場目線、誠実、押し売りしない |
| 映像 | 日本の住宅街、外壁塗装・リフォームの現場、スマホ音声入力、SaaS画面 |
| 文字 | 大きく、短く、1画面1メッセージ |
| NG | 細かすぎるUI、読めない表、過度な未来感、誇大な効果表現 |

## 5. 生成後チェック

- 冒頭3秒で何の動画か分かる
- テロップがスマホ画面で読める
- 「入力が苦手でも音声で作成」が伝わる
- 単価マスターまたは他システム出力のメリットが伝わる
- AIが確定する印象になっていない
- 「最後は人が確認」の前提が残っている
- 架空の導入実績、削減率、売上効果を出していない
- 原価、業者指示事項、内部情報非表示を主訴にしていない
